## 🚀 Cách sử dụng

1. **Ôn theo thứ tự**: Đọc từ file `01` → cuối cho mỗi framework
2. **Ôn theo chủ đề**: Chọn file theo chủ đề bạn cần ôn
3. **Ôn nhanh**: Đọc file tổng hợp (`ReactJS-Interview.md` / `NestJS-Interview.md`)
4. **Trước phỏng vấn**: Tập trung vào `tinh-huong-thuc-te.md` và tips
